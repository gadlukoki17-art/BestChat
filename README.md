# 💬 Kadea Chat

> A modern and responsive real-time messaging application built with HTML, Tailwind CSS and JavaScript.

Kadea Chat is a modern web-based messaging application that allows users to communicate through a clean, intuitive and responsive interface. Inspired by popular messaging platforms such as WhatsApp, it focuses on delivering a smooth user experience while showcasing modern front-end development practices.

This project was developed to strengthen my skills in JavaScript, REST APIs, DOM manipulation, responsive design and user interface development.

---

## ✨ Features

### 🔐 Authentication

- User registration
- User login
- Secure session management
- Automatic redirection when not authenticated

### 💬 Messaging

- Send and receive messages
- Automatic conversation refresh
- Conversation history
- Conversation preview with latest message
- Timestamp for each conversation
- Emoji picker
- Search conversations
- Search users
- Responsive mobile chat interface

### 👤 Profile

- View personal information
- Edit profile information
- Update full name
- Update bio
- Update email
- Local phone number management
- Avatar support

### ⚙️ Settings

- Dark Mode / Light Mode
- Language selection (English / Français)
- Modern settings panel

### 🎨 User Experience

- Clean and modern interface
- Mobile-first responsive design
- Desktop optimized layout
- Confirmation modal before deleting conversations
- Professional navigation experience


